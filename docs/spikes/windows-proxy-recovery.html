Spike · SPIKE-003 / Windows

Windows 系统代理恢复

结论：WinINet backend 能保存原始注册表类型与字节；应用被强杀后，独立 watchdog 可恢复原设置并通知 WinINet 刷新。

验证结果

项目结果
原始快照ProxyEnable、ProxyServer、ProxyOverride、AutoConfigURL、AutoDetect 与 Connections 原始值
隔离注册表测试值存在性、注册表类型和原始 bytes 精确往返
真实系统实验GitHub Windows 临时 runner 修改 HKCU Internet Settings 后强杀 owner
恢复结果watchdog 自动恢复，journal 删除，测试 1 passed / 0 failed
通知INTERNET_OPTION_SETTINGS_CHANGED + INTERNET_OPTION_REFRESH

剩余边界

  • macOS network service 快照与强杀恢复尚未验证，因此 SPIKE-003 仍为进行中。
  • 生产 journal 必须位于当前用户私有目录，并在发布前纳入权限与篡改测试。
  • 真实实验只在临时 CI runner 执行，不自动修改开发者机器的系统代理。
