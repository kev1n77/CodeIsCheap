Spike · SPIKE-003 / macOS

macOS 系统代理恢复

结论：networksetup backend 能按活动 network service 保存代理状态；应用被强杀后，root watchdog 可恢复原快照。

验证结果

项目结果
快照范围Web、Secure Web、PAC、Auto Discovery 与 bypass domains
服务范围只修改活动 network service；禁用服务保持不变
真实系统实验GitHub macOS 26 ARM64 临时 runner，以 root 执行 networksetup
恢复结果强杀 owner 后 watchdog 恢复完整快照并删除 journal
安全降级检测到认证代理时拒绝接管，避免无法恢复密码

生产要求

  • 桌面应用需通过受控 privileged helper 执行 networksetup/watchdog，不直接长期以 root 运行。
  • journal 必须位于私有目录；helper 只接受固定 backend 与严格校验后的设置。
  • 卸载、升级和多 network service 变化仍需进入后续故障注入矩阵。
