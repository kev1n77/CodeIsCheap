mitmproxy Sidecar 验证

结论：固定版本 sidecar 可以打包并真实转发请求；凭据在跨进程前删除，Prompt 与原始网络请求保持完整。

验证结果

项目结果

Windows 单文件制品29,510,084 bytes（约 28.1 MiB）

制品状态Spike 未签名；正式发布必须平台签名、固定 hash 并生成 SBOM

边界

  • IPC 只允许 loopback，并使用每次启动随机 token 的独立认证帧。
  • 只捕获明确 allowlist 中的 AI host；未知正文类型默认不发送。
  • 记录队列有界，IPC 故障只丢弃记录，不阻断目标网络。
  • 证书生命周期、系统代理恢复和生产签名不属于本 Spike。
