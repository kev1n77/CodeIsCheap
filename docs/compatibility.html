采集模式

Local Gateway

  • 客户端必须可配置 base URL
  • 不需要本地 CA
  • 支持流式、取消和错误透传
  • 推荐作为默认与兼容回退

Explicit Proxy

  • 应用必须使用系统/显式代理
  • HTTPS 需要信任本地 CA
  • 支持 HTTP/1.1、HTTP/2、SSE、压缩与 NDJSON
  • 不绕过 pinning，不覆盖 QUIC/TUN

已知边界

  • 证书固定、代理绕过、自带网络栈和 QUIC 可能无法捕获；UI 只给出低置信度诊断并建议 Gateway。
  • 进程归因依赖操作系统连接表；无法唯一匹配时保持 unknown，不做猜测。
  • 未知协议降级 Raw；产品不会恢复模型服务端隐藏 Prompt。
  • 新增 provider、端点或平台必须同时更新 Capture Policy、fixtures、能力矩阵和本页。
