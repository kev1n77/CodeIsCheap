产品定义

AI 输出由系统指令、历史消息、工具、文件、检索上下文和参数共同决定。CodeIsCheap 让这些客户端可观测内容变得透明。

目标用户

用户核心任务产品价值
AI 应用开发者定位上下文污染和版本差异原始证据 + 结构化差异
Prompt / Agent 工程师检查系统指令、工具和 token 分配Prompt Anatomy + 预算分析
QA / 安全评审检查敏感数据和越权工具规则、脱敏和审计证据

范围

要做

  • 捕获用户有权观察的 AI 请求。
  • 统一展示 Prompt、工具、参数与响应。
  • 搜索、比较、脱敏和导出。
  • 明确标记观察、推断与未知。

不做

  • 不恢复服务端隐藏 Prompt。
  • 不绕过证书固定或企业策略。
  • 不默认捕获全部 HTTPS 流量。
  • 不未经同意上传 Prompt。

MVP 功能

  1. Gateway 与显式代理捕获。
  2. 四类主流协议解析。
  3. Anatomy、Raw、Timeline 三种请求视图。
  4. 两请求结构化比较。
  5. 本地搜索与脱敏导出。
  6. 证书、代理的可靠安装与恢复。
