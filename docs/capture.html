范围和敏感字段由版本化 Capture Policy 驱动。sidecar 在 IPC 前执行一次，Rust Core 在进入持久化前再次执行；任一层判断越界即不记录。

Prompt IR

IR 是跨厂商统一的数据骨架。它保留消息顺序、工具 schema、原始 JSON 路径和厂商特有字段。

字段内容
instructions系统与开发者指令
messages用户、助手、工具消息
context检索文档、缓存上下文
tools工具定义、选择策略、调用结果
generationtemperature、max tokens 等参数
evidenceobserved、derived、inferred、unknown

明确边界

  • 应用忽略代理、使用 HTTP/3、独立证书库或证书固定时，代理可能无法捕获。
  • 服务端二次拼装的系统指令和 RAG 内容不可从客户端请求证明。
  • 流式响应丢失、取消或截断必须显示为不完整，不伪造最终结果。
  • 未知协议降级到 Raw，不阻塞请求转发。

解析器注册表按置信度选择候选并校验 Prompt IR。单个适配器错误或 panic 不影响其他候选；OpenAI Responses、Chat Completions 与 Completions 已由 golden fixtures 固定映射。
