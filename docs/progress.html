1. 更新规则

  1. 任务状态只使用：Not started In progress Blocked Done Deferred。
  2. 每个任务必须有 owner、目标日期、完成百分比和一句话备注。
  3. 百分比按可验收产物计算，不按已投入时间估算。
  4. 出现阻塞当天登记；解除后保留原因与处理结果。
  5. 每周更新一次里程碑、风险、质量指标与总体判断。

维护方式：直接编辑本 HTML 中对应表格。任务定义和完成标准以 开发计划 为准，避免在两个页面重复修改范围。
