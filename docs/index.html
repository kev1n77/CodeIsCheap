Product Design

CodeIsCheap

看清 AI 应用实际发出的请求，把杂乱 JSON 还原成可理解、可比较、可安全分享的 Prompt。

核心定位 CodeIsCheap 是本地优先的 AI 请求检查器，不是通用抓包器，也不承诺恢复模型服务端的隐藏 Prompt。

核心价值

捕获 AI 请求
→
识别厂商协议
→
还原 Prompt 结构
→
比较与分析
→
脱敏导出

关键结论

  • MVP 优先支持 OpenAI-compatible、Anthropic、Gemini、Ollama。
  • 首发采集方式为本地 API Gateway 与显式 HTTP(S) 代理。
  • 不同厂商统一转换为版本化 Prompt IR，同时保留原始证据路径。
  • 鉴权头永不落盘；所有导出先脱敏、后确认。
  • 桌面端采用 Tauri + React + Rust，TLS 捕获复用 mitmproxy sidecar。
