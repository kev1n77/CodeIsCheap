# CodeIsCheap 0.1.0

首个 Windows 与 macOS 桌面版本，面向需要检查 AI 应用真实请求的开发者。

## 主要能力

- 通过 Local Gateway 或显式 HTTPS Proxy 捕获目标 AI 请求。
- 结构化解析 OpenAI、Anthropic、Gemini 与 Ollama；未知或未声明能力保留为 Raw。
- 提供 Anatomy、Timeline、Raw、全文搜索和双请求 Compare。
- 使用 SQLCipher 本地加密，凭据在 sidecar IPC 前与持久化前双重删除。
- 支持单请求/批量脱敏导出、无 Prompt 支持包和签名应用内更新。

## 安全与恢复

- Windows named pipe 与 macOS/Linux Unix socket 均按当前用户隔离并核对 sidecar 进程。
- Proxy 模式使用事务快照与独立 watchdog；崩溃、退出和升级均恢复系统代理。
- 更新前生成加密恢复快照；主库失败时仅以只读模式开放历史搜索与导出。

## 已知限制

- 不绕过证书固定，也不承诺捕获绕过系统代理、QUIC 或自带网络栈的应用。
- Linux 仅作为开发与 CI 平台，不属于本版本 GA 安装范围。
- Mistral 端点可按策略捕获，但当前仅保证 Raw 证据，不声明结构化 Prompt IR。
- 产品不会恢复模型服务端隐藏的 system prompt。
